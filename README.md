# AutomationSTAR 2025

This repository contains materials related to my presentations at the [AutomationSTAR 2025](https://automation.eurostarsoftwaretesting.com/conference/2025) conference.

- The [Tutorial](Tutorial) directory contains examples created as part of the [Extending Robot Framework with Custom Libraries](https://automation.eurostarsoftwaretesting.com/event/2025/extending-robot-framework-with-custom-libraries/) tutorial.

- The [Talk](Talk) directory contains demos shown in the [Robot Framework: Latest News & Updates](https://automation.eurostarsoftwaretesting.com/event/2025/robot-framework-latest-news-updates/) track talk.
