# Talk: Robot Framework: Latest News & Updates

This directory contains demos shown as part of the [Robot Framework: Latest News & Updates](https://automation.eurostarsoftwaretesting.com/event/2025/robot-framework-latest-news-updates/) track talk as well as the presentation
slides.

Demos can in general be executed like `robot rf72.robot`. The ones needing
special configuration explain what is needed in their documentation.
